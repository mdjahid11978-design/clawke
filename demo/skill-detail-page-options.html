# 1Password CLI

Follow the official CLI get-started steps. Don't guess install commands.

## References

- `references/get-started.md` (install + app integration + sign-in flow)
- `references/cli-examples.md` (real `op` examples)

## Workflow

1. Check OS + shell.
2. Verify CLI present: `op --version`.
3. Confirm desktop app integration is enabled and the app is unlocked.
4. REQUIRED: create a fresh tmux session for all `op` commands.
5. Use `op read`, `op item get`, or `op run` only after confirming the user intent.

## Safety

Never print secrets directly unless the user explicitly asks for them.
